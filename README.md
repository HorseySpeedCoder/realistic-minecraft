# Realistic Minecraft (Browser)

A first-person Minecraft-like block game that runs directly in the browser.

## Run locally

```bash
python3 -m http.server 4173
```

Open <http://localhost:4173>.

## Controls

- `W A S D`: move
- `Shift`: move faster
- `Space`: jump
- Mouse: look around (after pointer lock)
- Left click: mine block
- Right click: place block
